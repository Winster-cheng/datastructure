#####2.1 选择排序：每一次从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置，然后，再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。以此类推，直到全部待排序的数据元素排完。
- 注意代码中min的作用有2个

#####2.2 插入排序 将一个数据插入到已经排好序的有序数据中，从而得到一个新的、个数加一的有序数据，算法适用于少量数据的排序，时间复杂度为O(n^2)。特点是:

- 第一层从i=1开始循环
- 第二层j=i从最右边的开始对比，一直向左移动直到左边的数值比该数值小
